# Jhanavi Putcha - Portfolio

An interactive portfolio website showcasing professional experience, projects, and skills.

## Prerequisites

- Node.js (v16 or later)
- npm or yarn package manager

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/jhanavi-portfolio.git
cd jhanavi-portfolio
```

### 2. Install dependencies

```bash
npm install
# or
yarn
```

### 3. Start the development server

```bash
npm run dev
# or
yarn dev
```

### 4. Open in browser

The application will be available at [http://localhost:5173](http://localhost:5173)

## Build for Production

To create a production build:

```bash
npm run build
# or
yarn build
```

The build output will be in the `dist` directory, which you can deploy to any static hosting service.

## Project Structure

- `src/components/` - UI components
- `src/App.tsx` - Main application component
- `src/main.tsx` - Application entry point
- `src/index.css` - Global styles
